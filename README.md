# 🍽️ Keurig Recipe Hub

Keurig Recipe Hub is a recipe browsing web application inspired by dishes shared by the **Keurig Food Society**.

The platform allows users to explore recipes created by members of the society, view ingredients and step-by-step cooking instructions, and access the original Instagram posts where the dishes were featured.

This project was built as a **portfolio project using HTML, CSS, and JavaScript**.

---

## 🌐 Live Website

🔗 https://isha-xo.github.io/keurig-recipe-hub/

---

## ✨ Features

- Browse dishes created by members of the **Keurig Food Society**
- View ingredients and step-by-step cooking instructions
- Search dishes instantly using the search bar
- Filter recipes by category (Desserts, Beverages, Main Course, Snacks)
- Random **Recipe of the Day** generator
- Direct link to the original Instagram post for each dish
- Responsive layout that works on desktop and mobile devices
- Displays number of recipes found based on search results

---

- **HTML5** – structure of the website

## 📖 How It Works

1. All recipe data is stored in **recipes.json**.
2. When the website loads, JavaScript fetches the data from the JSON file.
3. The left panel displays the list of dishes.
4. When a dish is clicked:
   - The ingredients and instructions appear in the right panel.
   - A link to the original **Instagram post** is shown.
5. The search bar filters dishes dynamically as the user types.
6. Category buttons allow users to filter recipes by type.
7. The **Recipe of the Day** button randomly selects and displays a recipe.
